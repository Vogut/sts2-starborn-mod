## 角色特色
主轴是**奇波**和**印记**两套机制，围绕它们构建了卡牌和遗物。奇波拥有自己的牌堆，并在每个回合结束时会随机打一张牌（不会吃大部分效果，不会计入出牌数）。印记参考原版，最大5层，3/5层时会在回合开始消耗特定数量的印记触发**调谐**/**超限**效果。（**随时可能更改**）

---

## 设计目标
- 目前总体上希望该角色偏向运转，平均卡组大小在25以下。
- 强度尽量合理（指我玩的爽）

## 属性效果
- 火调谐：使所有敌人获得当前层数的**灼烧**。  火超限：使所有敌人获得2倍当前层数的**灼烧**。
- 水调谐：获得2倍当前层数的**泉涌**。  水超限：获得2倍当前层数的**泉涌**，使所有敌人获得1层**淹没**。
- 木调谐：获得当前层数的**格挡**。  木超限：获得2倍当前层数的**格挡**，如果本场战斗第一次超限回复当前层数的生命。
- 光调谐：使所有敌人获得2x当前层数的**暴晒**。  光超限：使所有敌人获得2x当前层数的**暴晒**。造成5点攻击伤害。
- 风调谐：抽1张牌。  风超限：抽1张牌。获得1点能量。
- 冰调谐：使敌人获得当前层数的**结冰**。  冰超限：使敌人获得2倍当前层数的**结冰**。

## 能力列表
| 能力 | 效果 |
|------|------|
| 变大（Enlarge） | 每层使体型变大10%。|
